# 문제 해결의 절차   
1. 문제를 정확히 이해
2. 문제를 해결하는 알고리즘을 개발
3. 알고리즘이 문제를 해결한다는 것을 증명
4. 알고리즘이 제한시간내에 동작한다는 것을 보임
5. 알고리즘을 코드로 작성

## 시간복잡도   
✔️ 알고리즘이 대략 몇 개의 명령을 수행하는지    

## 완전 탐색(Brute-Force)      
✔️ 가능한 모든 경우를 시도해 보는 것   
- 가능한 모든 경우를 전부 고려해도 괜찮을 경우에는 단순히 모든 경우를 고려함으로써 문제를 해결
- 문제가 주어지면 무조건 완전 탐색법으로 먼저 시도해야 함
